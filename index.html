TO STUDY

  • Gsap

    GSAP는 화려하고 인터랙티브한 화면을 쉽고 빠르게 만들 수 있는 강력한 도구입니다. 앞으로 GSAP의 다양한 기능을 익혀 복잡한 애니메이션도 효율적으로 구현하고 프로젝트에 적극 활용할 수 있도록 꾸준히 학습하겠습니다.

  • Vanilla JS

    React, Vue.js 등 JavaScript 프레임워크/라이브러리 기초 이해를 위해 Udemy에서 바닐라 JavaScript 강의를 수강하고, jQuery로 작업했던 프로젝트를 JavaScript로 똑같이 구현하는 것을 목표로 하고 있습니다.

  • Three.js & Canvas

    인터랙티브한 화면을 위해 Three.js 강의를 통해 웹 브라우저에서 3D 그래픽을 렌더링하기 위해 사용되는 JavaScript 3D 라이브러리 Three.js를 학습하고 있습니다.

  • REACT

    프론트엔드 개발자로 성장하기 위해 React는 필수적인 기술이라고 생각합니다. 현재의 웹 개발 스킬을 더욱 향상시키고 사용자에게 편리하고 직관적인 웹 경험을 제공하는 개발자가 되겠습니다.

  • TypeScript

    React 프로젝트의 안정성과 가독성, 코드 품질을 높이기 위해 TypeScript를 학습하고, 그 장점을 활용하여 높은 퀄리티의 React 프로젝트를 만들어가도록 하겠습니다.

JavaScript 프레임워크, 라이브러리, CSS4와 같은 새로운 웹 기술들을 빠르게 학습하고 사용자 경험에 대한 최신 트렌드를 한발 앞서서 받아들이는 퍼블리셔가 되고자 합니다.
